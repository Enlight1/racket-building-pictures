Building Pictures with Racket

By Rowan Weir, Daniel Barnas, Allan Schougaard

Introduction

Dr. Racket provides many tools and primitives to allow you to design ever more complex and elaborate illustrations without needing to source them from elsewhere. We will begin by introducing the simplest primitives and build up from there.

Simple Shapes

By Rowan Weir, Daniel Barnas, Professor Allan Schougaard One of the easiest shapes to draw in Dr. Racket is the circle. It's primitive requires you to specify the radius of the circle (in pixels), the mode ("solid" or "outline"), and the color, in that order. (circle radius mode color) === image Let us construct our first shape in Dr. Racket using the following: (circle 25 "solid" "red") As expected, the resulting image is that of a solid red circle of radius 25 pixels. If we switch the mode to "outline," we get the following outline of a circle: (circle 25 "outline" "red") Try tinkering with the radius and notice the effect on the circle's size. Feel free to play around with the color as well. The simplicity of circles is that their radius is the same in all directions. What if instead we'd like to draw an elongated circle, or an ellipse? The ellipse function is very similar to the circle, except that now we need to specify a width (horizontal length) and a height (vertical length). (ellipse width height mode color) Let us type in the following to create our first ellipse: (ellipse 50 25 "solid" "pink") Can you tell which argument corresponds to the width? Which argument corresponds to the height? How do the width and height of this ellipse compare to the circles we created above? Why do you think the circles are so much larger?

Polygons

We will now go over some of the more popular shape primitives available in Dr. Racket and how to use them.

Squares

The easiest polygon to draw and visualize is the square. It takes a numerical value for the side-length followed by the usual mode and color arguments from the "circle" and "ellipse" functions above. (square side-len mode color) (square 25 "solid" "blue") (square 25 "outline" "blue") How does a square of side length 25 compare to a circle of radius 25? Which would you expect to be larger? Why?

Rectangles

The next logical shape to draw is the rectangle. By now the general outline for the functions used to define shapes should be becoming familiar. (rectangle width height mode color) (rectangle 50 25 "solid" "green") Could you think of a way to use the rectangle function to design square identical to the blue squares we drew above?

Triangles

You may recall from your high school geometry class that there are multiple types of triangles depending on how many side-lengths were equivalent. For the purposes of simplicity, we will only be going over equilateral triangles as well as right triangles. For more information on triangles and how you would go about designing more involved triangles, we invite you to read through the Dr. Racket documentation on triangles. As you may recall from high school, equilateral triangles are those whose three sides are of equal length, and whose three angles, by extension, are also congruent. Consequently, to draw an equilateral triangle, it is enough to simply specify the desired side length, followed by the usual mode and color arguments. (triangle side-length mode color) (triangle 25 "solid" "yellow") (triangle 25 "outline" "yellow") The next simplest triangle to construct is a right triangle, which as you may recall, is any triangle that contains a 90 degree angle. To construct a right triangle, you need to provide the two side-lengths of the sides exactly adjacent to the right angle. (right-triangle side-length1 side-length2 mode color) (right-triangle 50 25 "solid" "orange") Again, we invite you to experiment with the various triangle definitions such as isosceles-triangle or triangle/sss. Each of these allows you to specify certain side-lengths and angles, and the variety is virtually limitless.

Stars

Dr. Racket also provides functions to design various star designs. The simplest star available is the traditional five-pointed star that is available by calling the star function. (star side-length mode color) Here is an example: (star 25 "solid" "purple") The radial-star function allows for even more creativity, at the cost of a more complicated function call. (radial-star point-count inner-radius outer-radius mode color) The point-count is an integer value to determine how many points you'd like your star to have. The inner-radius determines the radius of the body of the star, while the outer radius determines the radius of the tips of the points of the star. The mode and color arguments are the same as for all the shapes above. (radial-star 15 50 25 "solid" "violet")

Regular Polygons

If you'd like to construct shapes with more that three of four sides, Dr. Racket makes it easy with the regular-polygon function. With this function, you can easily construct anything from pentagons, to octagons, to regular polygons with any number of sides, and of any side length. (regular-polygon side-length side-count mode color) (regular-polygon 20 8 "solid" "palegreen") Coincidentally, there is one more type of star we can construct while on the topic of regular polygons, and that is the star-polygon. The star-polygon is almost exactly the same as a regular polygon, except that it takes one more argument, the step-count. Instead of each vertex connecting to the one next to it, it instead connects to the vertex that is step-count vertices away, creating a star-like design. (star-polygon side-length side-count step-count mode color) Here's an example: (star-polygon 50 10 3 "outline" "indigo")
